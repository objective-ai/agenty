  <files>src/app/page.tsx</files>
  <action>
CRITICAL: Read BRANDING.md and AGENTS.md before writing any UI. Follow Adventure Navy design system strictly.

Replace the current root page.tsx with the PIN login screen. This is a client component (`'use client'`).

**Layout:**
- Full viewport, Adventure Navy background (#050B14)
- Cooper's tactical greeting at top: "Ready for your mission, Agent?" — styled in Cooper blue (#3B82F6)
- A 6-digit PIN display (shows dots/circles for entered digits, empty circles for remaining)
- A numeric keypad (0-9) arranged in a phone-style grid (1-2-3 / 4-5-6 / 7-8-9 / clear-0-backspace)
- Each key: chunky 2px border, deep shadow, min 64-72px for a kid, Cooper blue hover glow

**State management:**
- `pin`: string state (max 6 chars)
- `kidEmail`: loaded from localStorage on mount (key: `agenty_kid_email`)
- `isLoading`: boolean for submission state
- `error`: string | null for error display
- `lockout`: { unlockAt: string, remainingSeconds: number } | null
- `attemptsRemaining`: number | null

**PIN entry flow:**
1. Kid taps number keys -> digits fill the PIN display (left to right)
2. When 6 digits entered -> auto-submit (call `loginWithPin(kidEmail, pin)`)
3. On success -> `router.push("/bridge")`
4. On `wrong_pin` error -> shake the PIN display, show "Wrong code, Agent!" with attempts remaining, clear PIN
5. On `tactical_lockdown` error -> show Cooper's lockout message: "Tactical Lockdown! Base is secured." with countdown timer
6. Backspace key removes last digit, Clear key resets all

**Lockout countdown:**
- When lockout state is set, start an interval that counts down `remainingSeconds`
- Display as MM:SS format
- When timer reaches 0, clear lockout state and allow retries

**Styling:**
- Use Tailwind classes that match Adventure Navy: bg-[#050B14], border-2 border-white/10, shadow-lg
- Cooper blue (#3B82F6) for accents, hover states, active states
- PIN display dots: filled = Cooper blue, empty = white/20 opacity
- Error messages: red-tinted but still game-flavored
- Lockout message: amber/orange tint, Cooper delivering the message
