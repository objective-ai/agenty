# AGENTY: BRANDING & DESIGN SYSTEM

## The Core Vibe
- **Target Audience:** 9-to-11-year-old gamer. 
- **Aesthetic:** "Adventure RPG" meets "Tactile Tycoon". 
- **Strict Prohibition:** NO corporate SaaS designs. NO flat white cards. NO default Shadcn/Material UI styling. 
- **Brand Identity** The "AI" in the title represents the high-tech, futuristic nature of the companions. UI elements should reflect this with holographic effects and digital "data" glows.

## The Color World
- **Background:** Deep Adventure-Navy (`#050B14`)
- **Surfaces/Cards:** Midnight Slate (`#0A1423`)
- **Typography:** Cream/Parchment for text (`#F8F9FA` or similar), never pure harsh white.
- **Accents:** Gold coin shimmer, Torch-amber glow, Emerald for success/XP.

## The Component DNA ("Loot Chest Feel")
- All cards and main containers must have depth.
- **Borders:** Chunky `border-2` (neo-brutalism edge, but rounded).
- **Corners:** Friendly `rounded-2xl` or `rounded-3xl`.
- **Shadows:** Pronounced drop shadows (`shadow-lg` or custom glows) that feel tactile, like physical objects you can tap on an iPad.

## The Companion Engine (Dynamic Color Lanes)
The UI accent colors must shift dynamically based on which AI companion is currently active. 
When rendering a companion's specific UI or chat, use their signature color for glows, borders, and accents:
- **Cooper:** Strategic Cobalt (`#3B82F6`)
- **Arlo:** Workshop Orange (`#F97316`)
- **Minh:** Jade Green (`#10B981`)
- **Maya:** Storyteller Violet (`#8B5CF6`)
